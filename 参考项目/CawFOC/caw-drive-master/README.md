# CawDrive - 无刷电机驱动器

---

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/H2H3PQZVW)

# 🚀 简介

- 支持 24 电压输入

- 当前版本支持 SPI，串口和 CAN FD 总线通讯，支持 TIM PWM 刹车功能

- 主控芯片：STM32G474，电机驱动芯片为 DRV8323RSRGZR

- 板载磁编码器：AS5047P

- 支持一路NTC采样

- 支持 6xPWM 和 3xPWM 的控制
